# Healthboard

Сервис для проверки веб-сервисов указанных в фаловом конфиге

## Сборка и запуск

`$ make`

`$ ./healthboard`

`$ ./healthboard -h` - Для получения справки

## Обновление списка серверов без перезапуска сервиса

`$ ./updateConfig.sh`

## Веб-интерфейс
